Take the Microsoft Corporate Challenge! - An Overview of Microsoft Graph

Microsoft Graph is the gateway to data and intelligence in Microsoft 365. It provides a unified programmability model that you can use to access the tremendous amount of data in Microsoft 365, Windows 10, and Enterprise Mobility + Security. Come learn about Microsoft Graph and various developer tools you can use to utilize the wealth of data in Microsoft Graph to build apps for organizations and consumers that interact with millions of users. Join the Microsoft corporate challenge to build innovative solutions!

Time: Friday, 7:00PM EST
Presenter: Beth Pan

Practical things in tech I know now and wish I knew in college

Hacking Bluetooth for Fun and Profit

Prof. Zhiqiang Lin will introduce the security and privacy issues surrounding Bluetooth protocols and implementations. Participants will spend most of their time in a hands-on activity involving capturing, analyzing, and decrypting Bluetooth packets.

Participants will also be introduced to the exploreCSR program @ OSU (supported by Google Research), which encourages undergraduates -- especially students from underrepresented groups -- to get involved in CS research; exploreCSR will have several events throughout the year.

Time: Saturday, 11:00AM EST
Presenters: Prof. Zhiqiang Lin, Dr. Yue Zhang, Prof. Mike Bond
